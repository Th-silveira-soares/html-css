Trabalhando com Listas

Listas Ordenadas

Para criar uma lista ordenada, utiliza-se a tag <ol>.

Dentro da tag, existem alguns parâmetros adicionais como o type que serve configurar que tipo de ordenamento terá a lista atual.

No caso a seguir, foi utilizado o type = "a", fazendo com que a lista tenha ordenamento alfabético minúsculo.

O código fica: <ol type="a">.

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Um exemplo de concatenação de listas:

  1. Item 1
  2. Item 2
    1. Item 2.1
    2. Item 2.2
    3. Item 2.3
  3. Item 3
  4. Item 4
  5. Item 5

Listas Não Ordenadas

Para criação de listas não ordenadas, se faz o uso da tag <ul>.

Semelhante as listas ordenadas, em ul também existe o type, no entanto, não existe ordenação específica, apenas formatos geométricos como: disc, circle e square.
